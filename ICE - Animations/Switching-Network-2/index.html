Circuit Switching

Circuit switching is a connection-oriented network that establishes a dedicated path between the sender and receiver, providing a guaranteed data rate.

Examples

  • Analog telephone network
  • Optical mesh network
  • Public Switched Telephone Network (PSTN)

Phases of Circuit Switching

Circuit Establishment

A circuit-switched network consists of a set of switches connected by physical links. A connection between two stations is a dedicated path made of one or more links. However, each connection uses only one dedicated channel on each link. Each link is normally divided into n channels by using FDM or TDM. .

Data Transfer

Data and voice signals are transferred once the circuit is established.

Circuit Disconnection

The circuit is disconnected when one user initiates the disconnection.

Characteristics

  • Connection-oriented process
  • Fixed amount of data transferred
  • Termination occurs when the connection ends

Advantages

  • Steady communication rate
  • No intermediate delays

Disadvantages

  • Underutilization of resources
  • High establishment time

Types of Switches

  • Space Division Switches
  • Time-Division Switches
